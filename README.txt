collective.sentry
=================

Sentry integration with Zope.

Requirements
------------

* Plone 5.2 (tested)
* Python 3.6+ (tested)

Installation
------------

Add `collective.sentry` to your buildout and re-run buildout. 

Configuration
-------------

Configure the Sentry DSN by setting the environment variable `SENTRY_DSN`.
